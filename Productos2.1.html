GENERADOR DE RAMPA POR DAC (ICA-006)
El presente generador de rampa es empleando como base para dar origen a una señal eléctrica que sirva de referencia, por ejemplo para ajustar el barrido de una TV.  Por lo que este proyecto puede ser empleado como base para otras aplicaciones y aunque se tiene un objetivo didáctico en la comprensión de cómo opera el DAC que tiene incorporado, también puede emplearse de manera totalmente profesional este circuito.
A este circuito también se le inyectar una señal binaria que puede provenir del puerto paralelo una PC, y convertirla a una señal analógica mostrando una magnitud correspondiente al valor binario de que se trate. Por medio de un selector se puede seleccionar de entre una rampa ascendente o una descendente.

SILBATO ULTRASONICO PARA ENTRENAR CANINOS (ICA-008)
Existe una gran cantidad de animales que tienen la capacidad de captar sonidos de frecuencias altas, fenómeno que le es imposible al oído humano. Pues bien aprovechando la característica del oído de los caninos que pueden recibir (escuchar) sonidos que se encuentran entre el rango de 30 Hz a 50 KHz (el rango del oído humano va de 20 Hz a 20 KHz), se pueden emplear sonidos que el ser humano no puede escuchar para por ejemplo llamar a una mascota o disuadir a un perro callejero de provocar una mordedura.
De acuerdo a lo mencionado anteriormente, este circuito tiene la posibilidad de dos alternativas de operación las cuales servirán de forma especifica tanto para “entrenar caninos” como para “evitar el ataque de un perro”, ambos servicios que ofrece este silbato ultrasónico trabajaran a partir de sonidos cuyos valores de frecuencia se encuentra por encima del rango audible del oído humano (ultrasonido).
